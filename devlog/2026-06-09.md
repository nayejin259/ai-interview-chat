## 배운 것
- `useRef` — DOM 요소 직접 참조 시 제네릭으로 타입 지정 필요 (`useRef<HTMLDivElement>(null)`)
- `useEffect` 의존성 배열은 반드시 배열 형태 (`[messages]`)
- React Strict Mode는 개발 환경에서 `useEffect`를 두 번 실행함 → ref 플래그로 방지 가능
- `useEffect` 안에서 async 함수는 내부 함수 정의 후 호출하는 패턴 사용
- `setMessages(prev => ...)` 함수형 업데이트를 써야 stale state 버그 없음
- Web Speech API — `webkitSpeechRecognition`, TypeScript에서 `declare var`로 타입 선언 필요
- Gemini API는 대화 history가 반드시 user role로 시작해야 함
- `try/finally` — 성공/실패 상관없이 반드시 실행할 코드에 사용
- 커스텀 훅 — `use`로 시작하는 함수, React 훅 사용 가능, return으로 외부에 노출
