## 배운 것
- SSR 환경에서 브라우저 전용 API(`webkitSpeechRecognition`, `localStorage`)는 클라이언트에서만 실행되는 시점(이벤트 핸들러, `useEffect`)에 호출해야 함
- `useEffect(async () => {})` 불가 — async 함수는 Promise를 반환하는데 React는 cleanup 함수나 undefined만 허용함. 내부 async 함수를 정의하고 즉시 호출하는 패턴 사용
- `useEffect`를 쓰는 이유 — 컴포넌트 렌더링 시 직접 함수를 호출하면 state 변경 → 리렌더링 → 무한 루프. `[]` 의존성 배열로 최초 1회만 실행
- `useSearchParams()`는 반드시 `<Suspense>`로 감싸야 함 (Next.js App Router)
- Gemini 멀티모달 — PDF를 base64로 변환해 `inlineData`로 전달 가능
- `responseMimeType: 'application/json'` — Gemini가 JSON만 반환하도록 강제
- `FormData.append('file', file)` — 파일 업로드 시 `Content-Type` 헤더 직접 설정하면 안 됨 (브라우저가 자동으로 boundary 포함해서 설정)
